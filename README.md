# MagicalPixi
A website tool  for creating html5 game with pixi.js 



快速创建一个组件

> ./g-component.sh 组件名

快速接一个redux数据

> ./redux-new.sh 数据key名

生产环境部署

> ./deploy.sh

启动

> npm start
