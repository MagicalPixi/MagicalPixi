# MagicalPixi
A website tool  for creating html5 game with pixi.js 

启动

npm start

计划,优先级 p0>p1>p2:

  - p2 游戏工程结构和模版
  - p1 素材库 - 多个工程恭共享  
  - p0 MovieClip预览,并收入‘素材库’
  - p0 Sprite预览，并收入‘素材库’
  - p3 Stage新建-切换
